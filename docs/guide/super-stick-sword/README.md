# Super Stick Sword - 超级木剑

一个简易插件，灵感来源自模组Super Stick Sword

## ⚔️ 插件特色

**最后的压缩木剑拥有100点攻击力(50♥)**

类似于压缩木剑模组，需要玩家手动合成8重压缩木棍才可以合成超级~~母舰~~木剑。

## 🎮 游戏玩法

1. **收集木棍** - 从树叶获得或制作
2. **逐级压缩** - 按照合成表进行8重压缩
3. **制作超级木剑** - 用最终的压缩木棍制作
4. **享受强大威力** - 100点攻击力的超级武器
