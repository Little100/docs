## 🎮 主要功能

### 手动使用
通过不同等级的材料合成超级农事锄头：

| 物品名称 | 对应物品 | 对应最大耕地范围 |
|---------|---------|----------------|
| I级木锄 | 任意木板x8 | 1x1 |
| II级木锄 | 任意原木x1 | 1x1 |
| I级石锄 | 原石x8 | 3x3 |
| II级石锄 | 石头x1 | 3x3 |
| I级铁锄 | 铁锭x8 | 5x5 |
| II级铁锄 | 铁块x1 | 5x5 |
| I级金锄 | 金锭x8 | 7x7 |
| II级金锄 | 金块x1 | 7x7 |
| I级钻石锄 | 钻石x8 | 9x9 |
| II级钻石锄 | 钻石块x1 | 9x9 |

### 自动使用
- 用**II级锄头潜行+右键堆肥桶**即可放置自动农场
- 自动耕地、收割、收集和施肥
