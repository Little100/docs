 - ### [Modrinth](https://modrinth.com/plugin/emc)

# 2. 安装插件
## 你需要一个支持spigot插件的服务端
 - ① 开启服务器
 - ② 将下载的.jar文件放入plugins文件夹
 - ③ 享受
