# 此乃插件之号令，仅述其要者。
> 吾将以“有司”与“无司”二权分述之。

## 夫号令之主体
 - 可用者有二：
   - `/pe`
   - `/projecte`

# 有司之令
## reload
 - 用以重启插件（将重计EMC之值）。
## setemc
 - 用以设定物件之EMC值（君所持之物）。
 - 其法：`前令 + setemc <EMC值>`
## noemcitem
 - 用以查阅EMC为零之物。
## table
 - 可令转换台尽识万物。
 - 其法：`前令 + table learn`
## lang
 - 用以切换言语。
 - 其法：`前令 + lang <语言之档>`

# 无司之令
## pay
 - 用以支付EMC于玩家。
 - 其法：`前令 + pay <玩家之名> <EMC值>`
## bag
 - 用以查阅炼金术袋之列表。
 - 其法：`前令 + bag + list`
