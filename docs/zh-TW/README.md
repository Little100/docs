## **[Projecte-Plugin](/zh-TW/guide/)**
  - 一個靈感來源於等價交換模組 (ProjectE) 的插件，還原了絕大部分的功能和物品的喵。

此外掛短期內不再維護，精力不足，高中還住宿，所以暫時停止更新 qwp
:::

注意：此外掛目前有一個惡性 bug，玩家可以刷物品。建議暫時移除有關賢者之石合成其他物品的合成表。
:::
