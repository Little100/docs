# 一、手動使用
  - 用對應等級的物品即可合成超級農事鋤頭，對應物品如下列表：

| 物品名稱 | 對應物品 | 對應最大耕地範圍 |
| --- | --- | --- |
| I級木鋤 | 任意木板x8 | 1x1 |
| II級木鋤 | 任意原木x1 | 1x1 |
| I級石鋤 | 原石x8 | 3x3 |
| II級石鋤 | 石頭x1 | 3x3 |
| I級鐵鋤 | 鐵錠x8 | 5x5 |
| II級鐵鋤 | 鐵塊x1 | 5x5 |
| I級金鋤 | 金錠x8 | 7x7 |
| II級金鋤 | 金塊x1 | 7x7 |
| I級鑽石鋤 | 鑽石x8 | 9x9 |
| II級鑽石鋤 | 鑽石塊x1 | 9x9 |
  - 合成完畢後您可以使用潛行+切換副手以打開 GUI。

# 二、自動使用
 - 用**II級鋤頭潛行+右鍵堆肥桶**即可放置。
 - 效果如下：
   - 1. 自動耕地最大範圍（無視 Y 軸）。
   - 2. 自動收割農作物（無視 Y 軸）。
   - 3. 自動收集到附近容器（必須貼著堆肥桶）。
   - 4. 自動把附近容器內的骨粉用掉。
