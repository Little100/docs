# 配置文件
## 介紹
 - 你需要進入plugins/ProjectE目錄內 其中下面幾個文件是你可以查看的
```yaml
lang/en_us.yml
lang/zh_cn.yml
lang/lzh.yml
config.yml
accsspories.yml
command.yml
custommoditememc.yml
devices.yml
op_item.yml
recipe.yml
```

## 對應文件可以去側邊欄選擇
 - lang/*.yml --> 語言設置
 - config.yml --> 主配置文件
 - accsspories.yml,devices.yml,recipe.yml --> 合成表配置
 - command.yml --> 命令設置
 - custommoditememc.yml --> 自定義物品設置
 - op_item.yml --> op物品設置
